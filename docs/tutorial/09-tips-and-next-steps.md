### Daily Workflow

## OVERLAY

> **Start of day:**
> 1. Start Claude Code — context loads automatically
> 2. Ask "what were we working on?"
>
> **During work:**
> - `/nx:debug` when tests fail
> - `/nx:review-code` before committing
> - "Remember that we chose X because Y"
>
> **End of day:**
> - Git hooks auto-index on commit
